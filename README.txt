Démarrage de l'application : 

Nous avons utilisé l'IDE Pycharm pour développer notre application.
Elle comporte 2 librairies externes, notamment pour nous aider à tracer les graphiques.
Les 2 librairies sont : Numpy(v. 1.19.3) et Maplotlib(v. 3.3.2).
Il faut utiliser la version python 3.8 pour installer et compiler les librairies, 
nous avons eu des problèmes avec python 3.9

Procédure de lancement de l'application : 

	- Creer un nouveau projet sur votre ordinateur
	- Placer dans ce dossier le contenu de ce git : https://github.com/kechanut/OpenData
	- Décompresser l'archive 'jeu_donnees.zip'
		(Le format d'un des jeux de données à été modifié, notamment au niveau des dates,
		de ce fait vous devez utiliser ces sources et non télécharger les jeux directement sur le site)

	- Ouvrir l'invite de commande sur la racine du dossier
	- Faire la commande suivante : python -m pip install -U matplotlib
		Celle ci installera la librairie 'matplotlib'

	- OPTIONNEL : Vous pourriez avoir des soucis avec Numpy, dans ce cas effectuer la commande suivante : 'pip install numpy'
	- lancer l'application avec la commande 'py application.py'

Vous pouvez également lancer l'application directement depuis PyCharm
